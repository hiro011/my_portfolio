Applied Job Vacancy

Track Applied Job Vacancies

The job Vacancy web application is used to track the job vacancies you applied. It is made using HTML, CSS and PHP, and PHPMyAdmin as a local database. you can insert the job vacancies you applied and you can see, monitor, and track them. You can register as a new user, log in, insert a new application, delete, and search for the application you entered.

My role in the project

I made this project solo. so the whole project was made by me. I made this project when I was searching for a job as a fresh graduate, I applied to a lot of places to get more chances I sometimes forget where I applied so I wanted to track my applications. I started writing it in the notebook but I didn't like that so I told myself I consider myself a programmer a web and software developer so I have to make a web application that helps me with this task and thus I made this web application.

Problems encountered and their solutions

I didn't have big problems but I did have some issues with syntax, and others. Most problems were from PHP and database connection when inserting new data, and when I add authentication I wanted the user to have a profile, saving the profile image and having to show the user profile image and name in the home page was very challenging but with a few tricks and some StackOverflow help, I made it.
